将 OpenAI API 使用数据导出为 csv 的 chrome 扩展，可以将 csv 放在 excel 中进一步分析使用情况。

![csv](./docs/csv.png)

## 使用

1. 安装扩展
2. 打开 <https://platform.openai.com/>
3. 点击扩展图标
4. 选择时间范围下载 csv 文件
